# flaz.models.favelas

## Class Favelas

Representa um agregador de favelas.

Parâmetros
----------
f : str | Favela
    Pode ser um nome de distrito, nome de favela, uma instância Favela,
    ou uma lista de nomes para agregação.

Exemplos
--------
>f = Favelas("Vila Madalena")
>f.nomes
["São Remo", "Vila Nova Jaguaré", ...]
